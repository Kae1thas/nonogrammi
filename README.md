# Нонограммы Код Рисунка — React + Yandex Games

Полностью рабочий React-проект на Vite для публикации в Яндекс Играх.

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Готовая статическая версия появится в папке `dist`.

## Что уже встроено

- React-интерфейс в стиле опубликованного проекта с Судоку.
- Нонограммы 5×5, 7×7, 10×10 и 12×12.
- Прогресс и разблокировка уровней: следующий уровень открывается после 5 побед на предыдущем.
- Свободный режим без жизней: открывается после 5 побед на выбранной сложности.
- Жизни, монеты, подсказки, проверка, сдача, пауза.
- RU/EN и переключение темы.
- Подключение `/sdk.js` Яндекс Игр.
- `LoadingAPI.ready()` после готовности интерфейса.
- `GameplayAPI.start()` при первом игровом действии.
- `GameplayAPI.stop()` при паузе, победе, поражении и сдаче.
- Облачные сохранения через `player.getData()` / `player.setData()` с локальным fallback.
- Обработка `game_api_pause` / `game_api_resume`.
- Отключение системного скролла и pull-to-refresh, но с внутренней прокруткой `.app`, чтобы на Android не было мёртвых зон свайпа.

## Публикация в Яндекс Игры

Для загрузки в консоль Яндекс Игр используй содержимое папки `dist`, а не исходники проекта.
В этом архиве дополнительно есть отдельный zip `nonograms-yandex-upload.zip`, который уже собран из `dist`.
